# Data Preprocessing Pipeline - Titanic Dataset

This project demonstrates a complete data preprocessing workflow 
before training a machine learning model.

## Steps Covered
- Data Inspection
- Missing Value Treatment
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Encoding
- Feature Scaling

## Dataset
Titanic - Machine Learning from Disaster (Kaggle)

## Libraries Used
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
